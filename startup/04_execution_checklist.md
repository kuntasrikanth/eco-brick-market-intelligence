# Execution Checklist — TerraBlock Launch
### From Idea to First Brick Sold

---

## Phase 0 — Foundation (Month 1)
*Do these before spending any money on land or machines*

- [ ] Soil test: Collect soil samples from 3 sites in Ranga Reddy — send to lab for Atterberg limits + particle size (cost: ₹15K)
- [ ] Machine demo: Contact Hydraform India or CEB machine supplier — ask for on-site demo or visit existing CSEB plant
- [ ] Competitor visit: Drive to Hyderabad/Medchal area, visit 3–5 brick factories, understand buyer behavior
- [ ] 10 builder conversations: Talk to contractors/builders in Ranga Reddy — ask what they'd pay for a brick that cuts labour 30%
- [ ] Register company: Private Ltd or LLP — ₹8,000–15,000 via online portal
- [ ] Open business bank account: SBI or HDFC (required for MSME loan application)

## Phase 1 — Setup (Month 2–4)

**Legal & Compliance**
- [ ] MSME Udyam registration (free, online — msme.gov.in)
- [ ] GST registration (online — gst.gov.in)
- [ ] Factory / DIC registration (District Industries Centre, Ranga Reddy)
- [ ] Pollution clearance — Green Category (TSPCB — Telangana State PCB)
- [ ] Land lease agreement (get a lawyer to review, 3–5 year minimum lease)

**Financing**
- [ ] Apply MSME loan — SBI or HDFC MSME branch (bring: Udyam cert, projected P&L, KYC)
- [ ] Apply MUDRA Tarun — ₹10 lakh unsecured (same bank branch)
- [ ] Explore TIIC (Telangana Industrial Investment Corporation) subsidy
- [ ] Prepare project report (use the financial model — banks need this document)

**Land & Infrastructure**
- [ ] Finalize site (check: road access, power availability, no flood zone, soil test passed)
- [ ] Get building plan approved by Gram Panchayat / UDA
- [ ] Start civil construction: machine shed (40×60 ft), curing yard, office room, security cabin

## Phase 2 — Production Setup (Month 4–6)

**Machine & Equipment**
- [ ] Place machine order (allow 6–8 weeks delivery for Indian manufacturer)
- [ ] Get electrical connection from TSSPDCL (apply 2 months early)
- [ ] Install borewell (check groundwater depth in area first)
- [ ] Erect perimeter fencing + gate

**Trial Production**
- [ ] Hire 2 machine operators (can be from existing brick factory workers)
- [ ] Run 2-week trial production with local soil mix
- [ ] Adjust cement stabilizer % based on strength tests
- [ ] Produce 500 sample bricks — send 50 to NABL-accredited lab for BIS testing

**Sales Prep**
- [ ] Build 2 sample wall panels at the factory (one mortar, one dry-stack) — this is your showroom
- [ ] Create WhatsApp Business catalog with photos and price list
- [ ] Register on Google Maps / Google Business (free — critical for discoverability)
- [ ] Get 5–10 builder visits to the factory before launch

## Phase 3 — Launch & Ramp (Month 6–12)

**Sales**
- [ ] Offer first 2 pilot customers: supply at cost price (₹7/brick) in exchange for testimonial + referral
- [ ] Submit to PMAY approved supplier list (National Housing Bank)
- [ ] Approach 3 housing developers in Ranga Reddy with comparison cost sheet
- [ ] Attend one construction trade event in Hyderabad (CII Real Estate, CREDAI)

**Operations**
- [ ] Hire 1 sales executive (local, knows contractor community)
- [ ] Set up basic accounting (Tally or Zoho Books)
- [ ] Establish 3 soil suppliers — never depend on one source
- [ ] Sign first 3-month supply contract

**Quality & Certification**
- [ ] Apply for BIS IS 1725 certification once lab tests pass
- [ ] Start IGBC Green Materials documentation (takes 3–6 months)

## Month 12–18 — Scale Toward Break-Even

- [ ] Review utilization monthly — target 60% by Month 9, 75% by Month 12
- [ ] If demand exceeds supply, add 3rd shift before buying new machine
- [ ] Begin Kamareddy site scouting (visit District Collector office for land availability)
- [ ] Start talking to a second investor or bank for Phase 2 funding

## Red Flags to Avoid

1. **Don't buy land before soil test passes** — wrong soil = wrong product
2. **Don't order machines before financing is confirmed** — cash flow crisis in Month 1
3. **Don't skip the builder conversations in Phase 0** — you need to know price tolerance before building
4. **Don't try to sell to retail (B2C)** — go B2B only, at least for the first 2 years
5. **Don't underestimate TSPCB clearance time** — apply in Month 2, even if factory isn't ready
